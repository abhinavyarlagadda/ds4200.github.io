Project Introduction

This project will focus on air pollution datasets from the United States Environmental Protection Agency. By exploring and analyzing pollutant data across multiple geographic areas over time, we may be able to glean insights regarding overall trends, seasonal patterns, and the impact of human events, circumstances, and natural phenomena on public health. These insights may then be used in conjunction with public health and healthcare data to deliver informative suggestions to both the government and the public to aid in the improvement of public health and to help mitigate health risks during predictable events.

Data Description

The data is available through two methods: The data is downloadable from the U.S. Environmental Protection Agency (EPA) as .csv files, and can be further divided by pollutant, year, geographical area, and monitor site (categorical features) if a narrower scope is desired, and there is also an API (also from the EPA) which data can be extracted. There is significant variation in the number of rows available for a specific pollutant and geographical area, but each area will generally have somewhere between 400 and 900 rows for each pollutant (that is monitored) per year (estimated from available data for 2025). Significant features available in the dataset include the following: Date Daily Maximum Daily Air Quality Index (AQI) Pollutant Type Location (Cities, Counties, States) Daily Observation Count Monitor Site ID Depending on the scope of the finalized project, this database can later be used in conjunction with location-divided health data (available from the CDC website) to derive insights into correlations between public health and concentration of pollutants.

Visualization 1:

Description: This visualization compares Boston carbon monoxide levels and AQI over time, with federal holidays marked using vertical reference lines.

Takeaway: This chart helps show how carbon monoxide concentration and AQI change across the year and whether there are visible patterns around holidays.

Visualization 2:

Description: This box plot shows the distribution of AQI values across pollutant categories. It helps compare how air quality varies by pollutant, including differences in median AQI, spread, and potential outliers.

Takeaway: This visualization makes it easier to compare which pollutants tend to have higher or more variable AQI values. The box plot highlights overall spread and extreme values, helping show which pollutant categories may contribute more inconsistent air quality patterns.

Visualization 5:

Description: This visualization serves to show the concentrations of multiple parameters (pollutants) over all valid monitor sites across the United States over time. Disclaimer: not all monitoring sites will record the same parameters, and data for the last third of the year is sparse, if not absent due to the time needed to update the data on the public databases. Additionally, this visualization may take some time to load, given the size of the dataset.

Takeaway: Using this visualization, users can more easily observe significant changes across time, and can query specific dates or time periods, such as the days around July 4th (day=185), when carbon monoxide levels are expected to be higher. Tooltips included help give more specific information regarding location, date, and measurements.

Interaction Features

  • On-hover tooltips display monitor site, date, measurement specifics, and pollutant type.
  • The parameter dropdown allows to filter the dataset to view measurements for one specific pollutant at a time.
  • The time slider allows the user to observe, through colors, trends across periods of time, and pick up patterns temporally and geographically, with lower attention to any one particular point.
